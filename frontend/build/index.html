Getting Started

Okay, cool so let's cover a few basics on how this thing works...

Server Side Dependencies

These can be installed with npm install (duh.) You will need to do this the first time you clone/fork this repo

Client Side Dependency Management

These can be installed using bower install <package> --save. This will add them to bower.json as well as downloading them into the frontend/build directory.

Typically, you would normally have to copy these over and link to them etc. Fortunately {{ctrl.frameworkName}}, automatically adds them to vendor.js and builds the file for you. This means that all your dependences are just "there" ready to use.
